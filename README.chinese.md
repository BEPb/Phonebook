## 安装和运行
____
### 克隆储存库
 
```sh
$ cmd
$ git clone https://github.com/BEPb/Browser
$ cd Browser
```
 
### 安装依赖项
```sh
$ pip install -r requirements.txt
```

## 用法
启动程序界面：
```
python3 -m main
```
